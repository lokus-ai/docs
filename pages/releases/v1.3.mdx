## üöÄ Migration Guide

### Upgrading from v1.2

v1.3 is fully backward compatible with v1.2 workspaces.

**Recommended Steps:**

1. **Backup Your Workspace**
   ```bash
   # Create a backup before upgrading
   cp -r ~/Documents/Lokus ~/Documents/Lokus-backup-v1.2
   ```

2. **Download and Install**
   - Download v1.3 from [GitHub Releases](https://github.com/lokus-ai/lokus/releases/latest)
   - Install following platform-specific instructions

3. **Review OAuth Settings** (if using Gmail)
   - Navigate to Settings ‚Üí Integrations ‚Üí Gmail
   - Re-authenticate if prompted
   - Review OAuth permissions

4. **Explore New Features**
   - Try Bases by adding YAML frontmatter to notes
   - Enable MCP server in Settings ‚Üí AI ‚Üí MCP Server
   - Experiment with split panes (Cmd/Ctrl + \\)

5. **Report Issues**
   - [GitHub Issues](https://github.com/lokus-ai/lokus/issues/new)

**No Breaking Changes** - All your existing notes, settings, and workflows continue to work.
