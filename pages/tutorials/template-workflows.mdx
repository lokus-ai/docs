Automate your note-taking with powerful templates. Learn to create daily notes, meeting templates, project briefs, and more using variables, date functions, conditionals, and loops.

### Step 1: Your First Template

Let's create a simple template to understand the basics.

#### 1.1 Create a Template

1. Create a new note called `Template - Simple Note`
2. Add this content:

```markdown
---
name: Simple Note
description: A basic template example
type: template
---

# {{title}}

Created: {{date}}
Author: {{author}}

## Notes

{{content}}

---

Tags: #{{category}}
```

#### 1.2 Understanding Template Syntax

**Variables:**
```markdown
{{variableName}}
```

**Date Functions:**
```markdown
{{date.now}}
{{date.today}}
{{date.format('MMMM dd, yyyy')}}
```

**Filters:**
```markdown
{{text | upper}}
{{name | capitalize}}
{{content | truncate(100)}}
```

> **Note:**
**Info:** Templates are regular markdown files with special `{{syntax}}` that gets replaced when you use the template.


#### 1.3 Using Your Template

To use a template:
1. Create a new note
2. Click the template icon or use slash command `/template`
3. Select "Simple Note"
4. Fill in the variables
5. The template expands with your values


**Date Arithmetic:**
```markdown
{{date.addDays(7)}}                   ‚Üí 7 days from now
{{date.subWeeks(1)}}                  ‚Üí 1 week ago
{{date.addMonths(1)}}                 ‚Üí 1 month from now
```

**Date Properties:**
```markdown
{{date.getDay}}                       ‚Üí Day of month (1-31)
```

#### 3.2 Conditionals Syntax

**If statements:**
```markdown
{{#if condition}}
  Content when true
{{/if}}
```

**If-else:**
```markdown
{{#if condition}}
  Content when true
{{else}}
  Content when false
{{/if}}
```

**If-elseif-else:**
```markdown
{{#if condition1}}
  First condition
{{elseif condition2}}
  Second condition
{{else}}
  Default
{{/if}}
```

**Comparison operators:**
```markdown
{{#if count > 5}}
{{#if status == 'active'}}
{{#if priority != 'low'}}
{{#if score >= 80}}
```

#### 4.2 Loops Syntax

**Each loop:**
```markdown
{{#each array}}
  {{this}} - Current item
  {{@index}} - Index (0-based)
  {{@first}} - True if first item
  {{@last}} - True if last item
{{/each}}
```

**Range loop:**
```markdown
{{#each (range 1 11)}}
  Item {{this}}
{{/each}}
```

**Loop with conditionals:**
```markdown
{{#each items}}
  {{#if @first}}
    **First Item:** {{this}}
  {{elseif @last}}
    **Last Item:** {{this}}
  {{else}}
    - {{this}}
  {{/if}}
{{/each}}
```

### Step 5: Using Filters

Filters transform values inline.

#### 5.1 Text Filters

```markdown
{{name | upper}}                      ‚Üí JOHN DOE
{{name | lower}}                      ‚Üí john doe
{{name | capitalize}}                 ‚Üí John doe
{{name | capitalizeAll}}              ‚Üí John Doe
{{text | truncate(50, '...')}}       ‚Üí Truncated text...
{{title | slug}}                      ‚Üí url-safe-slug
{{text | trim}}                       ‚Üí Trimmed text
```

#### 5.2 Array Filters

```markdown
{{items | join(', ')}}                ‚Üí item1, item2, item3
{{items | first}}                     ‚Üí First item
{{items | last}}                      ‚Üí Last item
{{items | length}}                    ‚Üí Array length
{{items | sort}}                      ‚Üí Sorted array
{{items | reverse}}                   ‚Üí Reversed array
{{items | unique}}                    ‚Üí Unique items only
```

#### 5.3 Number Filters

```markdown
{{price | currency}}                  ‚Üí $1,234.56
{{number | round}}                    ‚Üí Rounded number
{{number | abs}}                      ‚Üí Absolute value
{{percent | percentage}}              ‚Üí 75%
```

#### 5.4 Date Filters

```markdown
{{date | dateFormat('MM/DD/YYYY')}}   ‚Üí 01/15/2024
{{date | timeAgo}}                    ‚Üí 2 hours ago
{{date | relative}}                   ‚Üí yesterday at 3:45 PM
```

### Step 6: Template Includes

Reuse template snippets across multiple templates.

#### 6.1 Create Reusable Snippets

```markdown
---

type: snippet
---

Created: {{date.format('MMMM dd, yyyy')}}

```

```markdown
---
name: Footer Snippet
type: snippet

#note #{{date.format('yyyy')}} #{{date.format('yyyy-MM')}}
```
