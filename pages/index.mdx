## Key Features

<div style={{ display: 'flex', alignItems: 'center', gap: '0.5rem', marginBottom: '1.5rem' }}>
  <Sparkles size={20} style={{ color: '#1a1a1a', strokeWidth: 1.5 }} />
  <span style={{ fontWeight: 'bold', fontSize: '1.1rem' }}>Powerful capabilities</span>
</div>

- **Task Management** - Integrated kanban system with `!task` and `@task` triggers
- **Command Palette** - Quick access to files, actions, and commands with ⌘K
- **Rich Markdown Editor** - TipTap-powered editor with live preview
- **Context Menus** - VS Code-style right-click menus for enhanced workflow
- **File Management** - Intuitive sidebar with file operations
- **Keyboard Shortcuts** - Native macOS shortcuts (⌘C, ⌘V, ⌘A, etc.)
- **Fluid UX** - Smooth animations and responsive design
- **Cross-Platform** - Built with Tauri for desktop performance

## What Makes Lokus Special?

### Modern Architecture
Built with cutting-edge technologies for optimal performance:
- **Tauri** - Rust-powered backend for speed and security
- **React 18** - Modern UI with concurrent features
- **TipTap** - Extensible rich-text editor
- **Vite** - Lightning-fast development and builds

### Intuitive Design
Inspired by the best note-taking apps like Notion and Obsidian, but simplified:
- Clean, distraction-free interface
- Familiar keyboard shortcuts
- Context-aware menus and actions
- Smooth, fluid interactions

### Developer-Friendly
Comprehensive testing and documentation:
- 102+ automated tests
- Full TypeScript support
- Extensible plugin architecture
- Open source and customizable
