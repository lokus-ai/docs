# Welcome to Lokus

Lokus is a sophisticated, local-first markdown note-taking application designed for power users who demand both flexibility and performance. Built with React and Rust, Lokus combines the extensibility of modern web technologies with the performance of native applications.

## What is Lokus?

Lokus is a next-generation knowledge management platform that brings together advanced rich text editing, wiki-style linking, powerful visualization, and AI integration into a single, cohesive application.

## Key Features

### Local-First Architecture
All your data stays on your machine with file-based storage, ensuring privacy and speed.

### Rich Text Editor
Full-featured markdown editor with wiki links, math equations, task management, and code highlighting.

### Knowledge Graph
Interactive 2D and 3D graph visualizations of your notes and connections.

### Plugin Ecosystem
VS Code-level extensibility with comprehensive plugin API, MCP integration, and marketplace for discovering extensions.

### Cross-Platform
Native performance on Windows, macOS, and Linux powered by Rust and Tauri.

## Core Capabilities

### Editor Features
- Full markdown support with custom TipTap extensions
- Wiki links with `[[Page Name]]` syntax and autocomplete
- Math equations using LaTeX/KaTeX (`$inline$` and `$$block$$`)
- Task lists with multiple statuses
- Code blocks with 100+ language syntax highlighting
- Tables, images, and smart paste functionality

### Knowledge Management
- Workspace-based organization
- Bidirectional link tracking
- Graph visualization (force-directed 2D/3D)
- Full-text search across all notes
- Template system for reusable structures
- Bases (database views) for structured data

### Extensibility
- **Plugin System** - VS Code-level extensibility framework
- **Sandboxed Execution** - Secure plugin runtime environment
- **MCP Integration** - Model Context Protocol server for AI (68+ tools)
- **Plugin APIs** - Editor, UI, Workspace, Canvas, Graph APIs
- **Plugin Marketplace** - Discover, install, and manage extensions
- **Hot Reloading** - Live plugin updates during development
- **Custom Extensions** - Add TipTap extensions, slash commands, layouts
- **Data Providers** - Integrate external services (Jira, Trello, etc.)

### Integrations & Experimental Features
- **Gmail Integration** - OAuth 2.0 authentication for email
- **MCP Server** - 68+ tools for AI assistants
- **Plugin Integrations** - Custom data providers (Jira, Trello, etc.)
- **Multimedia Library** - OCR, PDF processing, media gallery (experimental)
  - Tesseract OCR integration for text extraction from images
  - PDF text and metadata extraction
  - Media gallery with thumbnail generation
  - Support for images, PDFs, video, and audio files
  - Backend fully functional, UI integration in progress

## Technology Stack

### Frontend
- React 19
- TipTap 3.4.x (ProseMirror-based editor)
- Tailwind CSS
- Vite build system

### Backend
- Rust (latest stable)
- Tauri 2.0
- Tokio async runtime

### Visualization
- Sigma.js for network graphs
- Three.js for 3D rendering
- TLDraw for canvas/whiteboard
- D3-force for physics simulation
