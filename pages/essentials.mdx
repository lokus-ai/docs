## Quick Reference

### Most Important Shortcuts
- `⌘K` - Open Command Palette (your most important shortcut!)
- `⌘S` - Save current file
- `⌘N` - Create new file
- `⌘F` - Find in document

### Essential Actions
- **Create content**: Command Palette → "new file"
- **Find files**: Command Palette → type filename
- **Format text**: Use Markdown syntax or keyboard shortcuts
- **Save work**: Auto-saves as you type, `⌘S` for manual save
