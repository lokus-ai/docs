# AI Integration (MCP Server)

Lokus features a comprehensive Model Context Protocol (MCP) server that provides 68 sophisticated tools for AI assistant integration, transforming your workspace into an AI-enhanced knowledge management platform.

## Overview

The MCP server implements the industry-standard Model Context Protocol, enabling seamless integration with AI assistants like Claude. This unlocks powerful AI-driven workflows for note management, content analysis, and workspace optimization.

## Getting Started

### Prerequisites

To use AI integration, you need:
1. **Lokus desktop app** with MCP server enabled
2. **AI assistant** that supports MCP protocol (like Claude)
3. **Active workspace** with content to analyze

### Setup Process

1. **Enable MCP Server** in Lokus preferences
2. **Configure AI Assistant** to connect to Lokus MCP server
3. **Verify Connection** by running a simple tool test
4. **Start Using AI Tools** through your AI assistant

### Basic Usage

Once connected, you can ask your AI assistant to:

```
"Analyze my workspace and suggest improvements"
"Create a summary of my recent notes"
"Find broken links in my knowledge base"
"Organize my notes by topic"
"Generate a knowledge graph of my content"
```

## Tool Examples

### Creating and Managing Notes

```javascript
// Create a new note with template
await mcp.create_note({
  title: "Project Planning",
  template: "project",
  tags: ["planning", "work"],
  content: "Initial project outline..."
});

// Update existing note with backup
await mcp.update_note({
  path: "projects/planning.md",
  appendContent: "\n## New Section\nAdditional content...",
  createBackup: true
});
```

### Content Analysis

```javascript
// Analyze content for insights
const analysis = await mcp.analyze_content({
  path: "research/findings.md",
  analysisType: "comprehensive",
  extractKeywords: true,
  suggestConnections: true
});

// Generate content summary
const summary = await mcp.generate_note_summary({
  path: "meeting-notes.md",
  summaryLength: "medium",
  includeKeywords: true
});
```

### Workspace Operations

```javascript
// Get comprehensive workspace info
const workspaceInfo = await mcp.get_workspace_info({
  includeStats: true,
  includeHealth: true,
  includeSummary: true
});

// Backup workspace with compression
await mcp.backup_workspace({
  name: "weekly-backup",
  compression: true,
  includeMetadata: true
});
```

### Search and Discovery

```javascript
// Advanced content search
const results = await mcp.search_content({
  query: "machine learning",
  fileTypes: [".md", ".txt"],
  useRegex: false,
  maxResults: 20,
  contextLines: 3
});

// Find similar content
const similar = await mcp.find_similar({
  referencePath: "ai-research.md",
  similarityType: "content",
  threshold: 0.7,
  maxResults: 10
});
```

## Configuration

### Server Settings

The MCP server can be configured through:

```json
{
  "mcp": {
    "enabled": true,
    "port": 3000,
    "logLevel": "info",
    "maxConnections": 5,
    "timeout": 30000
  }
}
```

### Tool Permissions

Control which tools are available:

```json
{
  "tools": {
    "noteTools": true,
    "searchTools": true,
    "aiTools": true,
    "fileTools": true,
    "workspaceTools": true,
    "editorTools": true
  }
}
```

## Development

### Custom Tools

Developers can extend the MCP server with custom tools:

```javascript
class CustomTool {
  static getTools() {
    return [{
      name: "custom_analysis",
      description: "Perform custom content analysis",
      inputSchema: {
        type: "object",
        properties: {
          content: { type: "string" },
          analysisType: { type: "string" }
        }
      }
    }];
  }

  static async executeTool(name, args) {
    if (name === "custom_analysis") {
      // Custom implementation
      return await this.performCustomAnalysis(args);
    }
  }
}
```

## Troubleshooting

### Common Issues

**Connection Failed**
- Verify MCP server is running
- Check port configuration
- Ensure AI assistant MCP client is properly configured

**Tool Execution Errors**
- Check workspace path configuration
- Verify file permissions
- Review error logs for specific issues

**Performance Issues**
- Monitor memory usage
- Check for large file operations
- Consider enabling tool caching

### Debug Mode

Enable detailed logging:

```json
{
  "debug": {
    "mcp": true,
    "tools": true,
    "performance": true
  }
}
```
