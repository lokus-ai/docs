# Canvas & Whiteboard

**Version:** 1.3.3 | **Status:** Production Ready

The Canvas feature in Lokus provides an infinite digital whiteboard where you can visually organize ideas, create diagrams, and arrange notes spatially. Built on TLDraw, the canvas combines freeform drawing with structured note-taking, enabling visual thinking alongside traditional text-based workflows.

## What's New in v1.3.3

Version 1.3.3 brings critical reliability improvements, plugin integration, and powerful new drawing capabilities:

- **Reliability Improvements** - Rock-solid canvas experience:
  - Robust save/load system with atomic writes and error recovery
  - Auto-save every 30 seconds (configurable) with visual indicators
  - Crash recovery that automatically restores your work from the last save
  - Version history with timestamped snapshots for rollback
  - Corruption detection and automatic repair
- **Enhanced Drawing Tools** - Professional drawing toolkit:
  - Pen tool with pressure sensitivity and smoothing algorithms
  - Shape tools (rectangle, ellipse, triangle, diamond, hexagon, star)
  - Text tool with rich formatting and auto-sizing
  - Arrow and connector tools with auto-routing
  - Color picker with recent colors, custom palettes, and eyedropper
  - Line styles (solid, dashed, dotted) with customizable widths
  - Fill patterns and gradients
- **Advanced Organization** - Keep complex canvases organized:
  - Groups for bundling related elements together
  - Layers with show/hide, lock/unlock, and reordering
  - Alignment tools (left, center, right, top, middle, bottom)
  - Distribution (horizontal, vertical, even spacing)
  - Smart guides that snap to edges and centers
- **Rich Embeds** - Bring external content into your canvas:
  - Embed notes as interactive cards with live updates
  - Images with drag-and-drop, resizing, and cropping
  - PDF viewers embedded directly on canvas
  - Web page embeds with live preview
- **Professional Export** - Share your work in multiple formats:
  - PNG export at 1x, 2x, or 4x resolution with transparency options
  - SVG export for scalable vector graphics
  - PDF export with multi-page support and embedded fonts
- **Plugin Integration** - Extend canvas functionality:
  - Custom shape libraries via plugins
  - Export format plugins
  - Layout algorithm plugins
  - Third-party tool integrations

## Overview

Canvas allows you to move beyond linear note-taking, creating visual representations of ideas, brainstorming concepts, and organizing information spatially. It's perfect for mind mapping, diagramming, project planning, and visual collaboration.

### Key Features

- **Infinite canvas** with smooth panning and zooming
- **Drawing tools** including shapes, arrows, and freehand
- **Note embedding** directly on canvas
- **Collaborative editing** with real-time updates
- **Export options** to PNG, SVG, and PDF
- **Touch and stylus support** for tablets and drawing displays

### Plugins

Extend canvas functionality with the Lokus plugin system (v1.3.3+):

**Plugin Types:**
- **Custom Tools** - Add new drawing and selection tools
- **Shape Libraries** - Professional shape collections (flowcharts, UML, icons)
- **Export Formats** - Additional export formats (Figma, Sketch, Miro)
- **Layout Algorithms** - Auto-layout and smart positioning
- **Data Integrations** - Connect to external data sources

**Plugin API Example:**
```javascript
export function activate(context) {
  // Register custom shape
  context.canvas.registerShape({
    id: 'custom-cloud',
    name: 'Cloud Shape',
    icon: 'cloud',
    create: (x, y) => ({
      type: 'geo',
      x, y,
      props: { geo: 'cloud', w: 200, h: 100 }
    })
  });

  // Add export format
  context.canvas.registerExportFormat({
    name: 'Figma',
    extension: '.fig',
    export: async (canvas) => {
      // Export logic
      return figmaData;
    }
  });
}
```

**Popular Canvas Plugins:**
- **Flowchart Pro** - Professional flowchart shapes with auto-routing
- **Architecture Designer** - AWS, Azure, GCP icon libraries
- **UML Suite** - Complete UML diagram support
- **Chart Generator** - Generate charts from data
- **Miro Sync** - Import/export Miro boards
- **Excalidraw Import** - Convert Excalidraw files

**Plugin Permissions:**
Canvas plugins require these permissions:
- `canvas:shapes` - Register custom shapes
- `canvas:tools` - Add drawing tools
- `canvas:export` - Add export formats
- `canvas:data` - Access canvas data

## Troubleshooting

### Common Issues

**Issue:** Canvas lagging
**Solution:** Reduce quality, limit elements, close other panes

**Issue:** Elements not connecting
**Solution:** Ensure snap-to-objects enabled, hover on connection points

**Issue:** Can't export
**Solution:** Check permissions, reduce canvas size, try different format

**Issue:** Lost changes
**Solution:** Enable auto-save, check version history, recover from backup
