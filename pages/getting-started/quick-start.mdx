# Quick Start Guide

Get up and running with Lokus in minutes. This guide covers installation, initial setup, and your first steps with the application.

## Installation

### Download Lokus

1. **Visit the download page** at [lokus.ai/download](https://lokus.ai/download)
2. **Select your platform:**
   - **macOS:** Download the `.dmg` file
   - **Windows:** Download the `.exe` installer
   - **Linux:** Download the `.AppImage` or `.deb` package

### Install the Application

#### macOS
1. Open the downloaded `.dmg` file
2. Drag Lokus to your Applications folder
3. Launch Lokus from Applications

#### Windows
1. Run the downloaded `.exe` installer
2. Follow the installation wizard
3. Launch Lokus from the Start menu

#### Linux
1. Make the `.AppImage` executable: `chmod +x Lokus.AppImage`
2. Run the application: `./Lokus.AppImage`
3. Or install the `.deb` package: `sudo dpkg -i lokus.deb`

## First Launch

### Initial Setup

When you first launch Lokus, you'll see the welcome screen:

1. **Choose your workspace location** - Select where you want to store your notes
2. **Optional: Sign in** - Connect to your Lokus account for sync and cloud features
3. **Select a theme** - Choose light or dark mode

### Workspace Creation

If you don't have an existing workspace, Lokus will help you create one:

1. **Workspace name** - Give your workspace a meaningful name
2. **Location** - Choose where to store your files (default: `Documents/Lokus-Workspace`)
3. **Template** - Select a template or start blank

## Core Concepts

### Workspaces

A **workspace** is a self-contained collection of notes, files, and settings. Think of it as a project or knowledge domain.

- Each workspace has its own folder on your computer
- You can have multiple workspaces for different projects
- Switch between workspaces using the workspace selector

### Notes

**Notes** are markdown files that support rich formatting:

- Text formatting (bold, italic, headings)
- Lists and checkboxes
- Math equations with KaTeX
- Wiki-style links between notes
- Code blocks with syntax highlighting

### Organization

Lokus provides flexible organization:

- **Folders** - Traditional hierarchical structure
- **Tags** - Cross-cutting organization
- **Wiki Links** - Interconnected knowledge graphs
- **Canvas** - Visual mind maps and diagrams

## Your First Note

Let's create your first note:

1. **Click the "+" button** or press `Cmd/Ctrl + N`
2. **Choose "Note"** from the options
3. **Add a title** - Try "Getting Started with Lokus"
4. **Start writing** - The editor supports markdown

### Basic Formatting

Try these formatting options:

```markdown
# This is a heading

**Bold text** and *italic text*

- Bullet point lists
- [x] Completed task
- [ ] Incomplete task

[[Link to another note]]

Math: $E = mc^2$

```python
# Code blocks with syntax highlighting
def hello_world():
    print("Hello, Lokus!")
```
```

### Saving

Lokus auto-saves your work, but you can also:
- Press `Cmd/Ctrl + S` to save manually
- See the save indicator in the status bar

## Key Features to Explore

### 1. Wiki Links

Create connections between notes:

1. Type `[[` to start a wiki link
2. Type the name of another note
3. Press Enter to create the link
4. Use the graph view to see connections

### 2. Math Equations

Add mathematical expressions:

- **Inline math:** `$x^2 + y^2 = z^2$`
- **Block math:** `$$\int_{0}^{\infty} e^{-x} dx = 1$$`

### 3. Task Lists

Create interactive checkboxes:

```markdown
- [x] Install Lokus
- [x] Create first note
- [ ] Explore advanced features
- [ ] Set up synchronization
```

### 4. Split View

Work with multiple notes simultaneously:

1. Click the split view button in the toolbar
2. Drag a note to the second pane
3. Resize panes by dragging the divider

### 5. Search

Find content across your workspace:

1. Press `Cmd/Ctrl + F` for global search
2. Use the search bar in the sidebar
3. Filter by file type, tags, or date

## Customization

Learn the most useful shortcuts:

- `Cmd/Ctrl + N` - New note
- `Cmd/Ctrl + O` - Open note
- `Cmd/Ctrl + P` - Command palette
- `Cmd/Ctrl + \` - Toggle split view
- `Cmd/Ctrl + /` - Toggle comment

## Troubleshooting

### Common Issues

**Lokus won't start:**
- Check system requirements
- Try restarting your computer
- Reinstall the application

**Notes not saving:**
- Check disk space
- Verify file permissions
- Try creating a new workspace

**Sync not working:**
- Check internet connection
- Verify account credentials
- Try signing out and back in

**Performance issues:**
- Close unused notes
- Reduce workspace size
- Clear the application cache
