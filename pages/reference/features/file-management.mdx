# 📁 File Management

Intuitive file operations with a clean interface and multi-window support.

## Sidebar Navigation
- **File Tree**: Hierarchical view of your workspace
- **Quick Search**: Type to filter visible files
- **Drag and Drop**: Move files and folders
- **Expand/Collapse**: Navigate large folder structures

## File Operations
- **Create**: New files and folders
- **Rename**: In-place renaming with validation
- **Delete**: Safe deletion with confirmation
- **Move**: Drag and drop or cut/paste
- **Copy**: Duplicate files and folders

## Multi-Window Support
- **Workspace Windows**: Open multiple workspace windows simultaneously
- **Preferences Window**: Dedicated settings window with theme persistence
- **Window Management**: Independent window state and configuration
- **Cross-Window Sync**: Consistent theme and settings across all windows

## File Types
Lokus recognizes and handles various file types:
- **Markdown** (`.md`, `.markdown`) - Full editing support
- **Text** (`.txt`) - Plain text editing
- **Code** (`.js`, `.ts`, `.py`, etc.) - Syntax highlighting
- **Images** (`.png`, `.jpg`, `.gif`) - Preview and embedding

## File System Permissions
Enhanced workspace access with:
- **Secure File Access**: Proper file system permissions handling
- **Multi-Window Compatibility**: Consistent file access across all windows
- **Performance Optimizations**: Efficient file system operations
