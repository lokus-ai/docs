# 🔧 Customization

Personalize Lokus to match your workflow and preferences.

## Theme Support
- **Light Theme**: Clean, bright interface for daytime work
- **Dark Theme**: Easy on the eyes for long writing sessions
- **System Theme**: Automatically matches your OS preference
- **Cross-Window Sync**: Consistent theme across all windows

## Editor Settings
- **Font Family**: Choose your preferred typeface
- **Font Size**: Adjustable from 12px to 24px
- **Line Height**: Control line spacing for optimal readability
- **Letter Spacing**: Fine-tune character spacing

## Layout Options
- **Sidebar Toggle**: Show/hide file sidebar
- **Focus Mode**: Distraction-free writing environment
- **Typewriter Mode**: Keep current line centered
- **Window Management**: Customize multi-window behavior

## Plugin System
- **Modular Extensions**: Add functionality without core changes
- **Custom Plugins**: Create your own extensions
- **Theme Extensions**: Custom themes and styling options
- **Status Bar Plugins**: Enhance the status bar with custom components

## Preferences Persistence
- **Settings Sync**: Consistent preferences across application restarts
- **Multi-Window Support**: Independent settings per window when needed
- **Export/Import**: Backup and share your configuration
