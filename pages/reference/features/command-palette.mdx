# ⌘K Command Palette

The command palette is your gateway to quick actions and file navigation.

## Access
- **Keyboard**: Press `⌘K` (Mac) or `Ctrl+K` (Windows/Linux)
- **Menu**: View → Command Palette

## Features
- **File Search**: Quickly find and open any file in your workspace
- **Recent Files**: Access recently opened files
- **Quick Actions**: 
  - Save current file (`⌘S`)
  - Create new file
  - Create new folder
  - Open preferences
  - Toggle sidebar

## Template Integration
- **Insert Template**: Access the template library directly from the command palette
- **Template Search**: Quickly find and insert templates by typing
- **One-Click Insertion**: Insert templates with automatic variable processing

## Usage Tips
```
⌘K → Type filename → Enter (Open file)
⌘K → "new" → Enter (Create new file)
⌘K → "save" → Enter (Save current file)
⌘K → "template" → Enter (Browse templates)
```
