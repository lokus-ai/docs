# ⌨️ Keyboard Shortcuts

Native keyboard shortcuts for efficient workflow.

## Essential Shortcuts
| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Command Palette | `⌘K` | `Ctrl+K` |
| Save File | `⌘S` | `Ctrl+S` |
| New File | `⌘N` | `Ctrl+N` |
| Open File | `⌘O` | `Ctrl+O` |
| Close Tab | `⌘W` | `Ctrl+W` |

## Edit Shortcuts
| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Cut | `⌘X` | `Ctrl+X` |
| Copy | `⌘C` | `Ctrl+C` |
| Paste | `⌘V` | `Ctrl+V` |
| Select All | `⌘A` | `Ctrl+A` |
| Undo | `⌘Z` | `Ctrl+Z` |
| Redo | `⌘⇧Z` | `Ctrl+Shift+Z` |

## Search Shortcuts
| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Find | `⌘F` | `Ctrl+F` |
| Find and Replace | `⌘⌥F` | `Ctrl+Alt+F` |

## Template Shortcuts
| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Insert Template | `⌘K` → "template" | `Ctrl+K` → "template" |
| Template Search | `⌘K` → type template name | `Ctrl+K` → type template name |
