# Features Overview

Lokus provides a comprehensive set of features designed to enhance your note-taking and writing experience. This page provides an overview of all features organized by user journey and complexity.

## Essential Features
*Core functionality that every user should know*

### Writing & Editing
- **[Rich Text Editor](essentials/writing/editor-basics)** - Modern editor with live Markdown support
- **[Basic Formatting](essentials/writing/formatting)** - Headers, emphasis, lists, and more
- **[Markdown Support](essentials/writing/markdown-support)** - Full Markdown compatibility

### File Management
- **[File Operations](essentials/organization/file-management)** - Create, rename, delete, and organize files
- **[Workspace Navigation](essentials/organization/workspace-basics)** - Navigate your content efficiently
- **[Search & Discovery](essentials/organization/search-basics)** - Find content quickly

### Productivity Boosters
- **[Command Palette](essentials/productivity/command-palette)** - Quick access to everything (`⌘K`)
- **[Template Basics](essentials/productivity/templates-basics)** - Reusable content templates
- **[Essential Shortcuts](essentials/productivity/shortcuts-essentials)** - Must-know keyboard shortcuts

## New in Recent Updates

### Universal Template System
Enhanced template capabilities with:
- Dynamic variable substitution (`{{date}}`, `{{time}}`, `{{user}}`)
- JavaScript execution within templates
- Command palette integration for quick access
- Secure sandbox for safe script execution

### Enhanced Plugin Architecture  
Improved plugin development with:
- Full React component integration
- Status bar plugin support
- Inter-plugin communication
- MCP (Multi-Computational Platform) integration

### Modern Documentation
This documentation site itself is new, featuring:
- Progressive disclosure design
- User journey-based organization
- Comprehensive examples and tutorials
- Interactive components and better search
