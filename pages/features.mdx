# Features

Lokus provides a comprehensive set of features designed to enhance your note-taking and writing experience.

## ⌘K Command Palette

The command palette is your gateway to quick actions and file navigation.

### Access
- **Keyboard**: Press `⌘K` (Mac) or `Ctrl+K` (Windows/Linux)
- **Menu**: View → Command Palette

### Features
- **File Search**: Quickly find and open any file in your workspace
- **Recent Files**: Access recently opened files
- **Quick Actions**: 
  - Save current file (`⌘S`)
  - Create new file
  - Create new folder
  - Open preferences
  - Toggle sidebar

### Usage Tips
```
⌘K → Type filename → Enter (Open file)
⌘K → "new" → Enter (Create new file)
⌘K → "save" → Enter (Save current file)
```

## 🖱️ Context Menus

VS Code-style context menus provide quick access to relevant actions.

### File Context Menu
Right-click on any file in the sidebar to access:

- **Open** - Open the file in the main editor
- **Open to the Side** - Open in a new tab/split view
- **Open With...** - Choose application to open with
- **Reveal in Finder** - Show file in system file manager
- **Open in Integrated Terminal** - Open terminal at file location
- **Cut** (`⌘X`) - Cut file for moving
- **Copy** (`⌘C`) - Copy file for duplicating
- **Copy Path** - Copy file path to clipboard
- **Rename...** - Rename the file
- **Delete** - Move file to trash
- **Share** - Share file options

### Editor Context Menu
Right-click in the editor to access:

#### Edit Actions
- **Cut** (`⌘X`) - Cut selected text
- **Copy** (`⌘C`) - Copy selected text  
- **Paste** (`⌘V`) - Paste from clipboard
- **Select All** (`⌘A`) - Select all content

#### History
- **Undo** (`⌘Z`) - Undo last action
- **Redo** (`⌘⇧Z`) - Redo last undone action

#### Search
- **Find** (`⌘F`) - Find text in document
- **Find and Replace** (`⌘⌥F`) - Find and replace text

#### Commands Submenu
- **Command Palette** - Open command palette
- **Insert Table** - Insert markdown table
- **Insert Code Block** - Insert fenced code block
- **Insert Link** - Insert markdown link
- **Insert Image** - Insert image reference

#### Export
- **Export as Markdown** - Save as .md file
- **Export as HTML** - Convert to HTML
- **Import File...** - Import content from file

## 📝 Rich Text Editor

Powered by TipTap, our editor provides a seamless writing experience.

### Markdown Support
- **Headers**: `# H1`, `## H2`, `### H3`, etc.
- **Emphasis**: `**bold**`, `*italic*`, `~~strikethrough~~`
- **Lists**: Ordered (`1. item`) and unordered (`- item`)
- **Task Lists**: `- [x] completed` and `- [ ] todo`
- **Links**: `[text](url)` and `[[wiki links]]`
- **Code**: `inline code` and ` ```fenced blocks``` `
- **Tables**: Full markdown table support
- **Math**: LaTeX math expressions with `$inline$` and `$$block$$`

### Live Features
- **Auto-paste Processing**: Automatically converts pasted markdown
- **Smart Typography**: Auto-correct quotes, dashes, and ellipses
- **Syntax Highlighting**: Code blocks with language-specific highlighting
- **Link Preview**: Hover to preview link destinations
- **Image Embedding**: Drag and drop image support

### Performance Optimizations
- **Efficient Rendering**: Minimal re-renders with `shouldRerenderOnTransaction: false`
- **Lazy Loading**: Extensions loaded on demand
- **Memory Management**: Automatic cleanup of unused resources

## 📋 Task Management

Integrated task management system that seamlessly connects notes with visual kanban boards.

### Task Creation Widget
Create tasks directly in your notes using the `!task` trigger:
- **Natural Workflow**: Type `!task ` anywhere in your notes
- **Task Types**: Urgent (🔴), Question (🟡), In Progress (🔵), Todo (⚪)
- **Quick Input**: Tab to cycle types, Enter to create, Escape to cancel

### Task Import Widget  
Reference existing tasks using the `@task` trigger:
- **Smart Search**: Type `@task ` to search and import existing tasks
- **Keyboard Navigation**: Arrow keys to navigate, Enter to select
- **Cross-Reference**: Link related tasks across different notes

### Kanban Board
Visual task organization with dual views:
- **Mini Kanban**: Compact sidebar view for quick overview
- **Full Kanban**: Dedicated tab with drag-and-drop functionality
- **Status Columns**: Organize tasks by progress and priority
- **Interactive Elements**: Click tasks in notes to jump to kanban

### Key Benefits
- **Seamless Integration**: Tasks created in natural writing flow
- **Visual Organization**: Kanban board for task management
- **Cross-Referencing**: Link tasks between different notes
- **Status Tracking**: Visual progress with color-coded status indicators
